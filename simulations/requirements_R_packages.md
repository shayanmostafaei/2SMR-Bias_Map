# Required R packages

The current simulation scripts use some or all of the following R packages:

- MASS
- ggplot2
- dplyr
- tidyr
- cowplot
- mr.raps
- mr.simss
- MendelianRandomization
- Matrix
- doParallel
- foreach
- meta
- MRSamePopTest
- MRMix
- PCMR
- R2jags

Some scripts may additionally depend on JAGS being installed and accessible from R.
