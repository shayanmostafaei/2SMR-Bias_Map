# Figures

This folder contains manuscript-linked figures and panel outputs for the 2SMR bias map project.

## Structure

- `final/` — assembled manuscript-ready figures
- `cluster1/` — outputs for overlap / weak instruments / winner’s curse
- `cluster2/` — outputs for pleiotropy / LD architectures
- `cluster3/` — outputs for population mismatch / family structure

## Main manuscript figure

The current main figure is a merged multi-panel figure covering three major bias domains in two-sample Mendelian randomization:

- A–C: overlap, weak instruments, and threshold-based instrument selection
- D–H: pleiotropy and LD across five architectures
- I–J: population mismatch and family-structure confounding
